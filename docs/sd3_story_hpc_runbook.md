# SD3 Medium Storyboard LoRA HPC Runbook

This runbook adds a parallel SD3 Story workflow. It does not replace the
existing Qwen independent-panel pipeline.

## 1. Environment

```bash
export PROJ_ROOT=/data/home/$USER/code/DSAA2012/Project
export VENV_ROOT=/data/home/$USER/code/DSAA2012/venvs/sd3story
export ASSET_ROOT=/data/scratch/$USER/dsaa2012_sd3_assets
export HF_HOME=$ASSET_ROOT/hf_cache

cd "$PROJ_ROOT"
mkdir -p outputs/sd3_story/logs "$ASSET_ROOT/hf_cache"
python -m venv "$VENV_ROOT"
source "$VENV_ROOT/bin/activate"
pip install --upgrade pip
pip install -r requirements.txt
```

If the cluster provides modules, load Python 3.10 and CUDA first.

```bash
module purge
module load python/3.10
module load cuda/12.1
```

Install the official Diffusers training scripts locally:

```bash
git clone https://github.com/huggingface/diffusers.git
pip install -e "./diffusers[torch]"
```

## 2. Hugging Face Access

Stable Diffusion 3 Medium is gated. Accept the license on Hugging Face, then:

```bash
huggingface-cli login

python - <<'PY'
from huggingface_hub import model_info
print(model_info("stabilityai/stable-diffusion-3-medium-diffusers").modelId)
PY
```

## 3. Prepare Prompts And Storyboard Data

```bash
python scripts/sd3_build_story_prompts.py \
  --input data/task_a \
  --out-dir data/sd3_story/validation_prompts
```

Prepare local task-derived training data:

```bash
python scripts/sd3_prepare_storyboard_data.py \
  --sources local \
  --out-dir data/sd3_story/train_storyboard \
  --resolution 768 \
  --clean
```

Prepare mixed data with external sources. Use this only on a machine with
network access and enough cache space:

```bash
python scripts/sd3_prepare_storyboard_data.py \
  --sources local comix flintstones \
  --out-dir data/sd3_story/train_storyboard \
  --resolution 768 \
  --max-comix 80 \
  --max-flintstones 80 \
  --dataset-cache-dir $ASSET_ROOT/hf_cache \
  --clean
```

If a dataset split name differs on the cluster cache, override it with
`--comix-split` or `--flintstones-split`.

The current training commands use Diffusers DreamBooth LoRA with one
`instance_prompt` per run. The `.txt` captions and `metadata.jsonl` generated by
the preparation script are for data auditing, report tables, and future trainer
extensions; they are not consumed as per-image captions by these shell scripts.

## 4. Storyboard LoRA Training

First smoke test:

```bash
MAX_TRAIN_STEPS=20 RANK=4 USE_8BIT_ADAM=0 bash scripts/sd3_train_storyboard_lora.sh
```

Full training:

```bash
RANK=8 MAX_TRAIN_STEPS=800 bash scripts/sd3_train_storyboard_lora.sh
```

Slurm:

```bash
sbatch --export=ALL,ROOT_DIR=$PROJ_ROOT,VENV_PATH=$VENV_ROOT/bin/activate,RANK=8,MAX_TRAIN_STEPS=800 \
  scripts/sd3_train_storyboard_lora.sbatch
```

Rank/step ablation:

```bash
for rank in 4 8 16; do
  for steps in 100 800 1200; do
    sbatch --export=ALL,ROOT_DIR=$PROJ_ROOT,VENV_PATH=$VENV_ROOT/bin/activate,RANK=$rank,MAX_TRAIN_STEPS=$steps,OUTPUT_DIR=artifacts/loras/sd3/storyboard_lora_rank${rank}_steps${steps} \
      scripts/sd3_train_storyboard_lora.sbatch
  done
done
```

## 6. Inference Matrix

Base SD3 one-shot:

```bash
python scripts/sd3_infer.py \
  --prompts data/sd3_story/validation_prompts/prompts_3scene.jsonl \
  --out-dir outputs/sd3_story/base_sd3 \
  --width 1024 --height 768 --num-candidates 4
```

Base SD3 independent scenes:

```bash
python scripts/sd3_infer.py \
  --prompts data/sd3_story/validation_prompts/prompts_independent_3scene.jsonl \
  --out-dir outputs/sd3_story/ablation_independent_scenes \
  --width 512 --height 512 --num-candidates 4

python scripts/sd3_compose_independent.py \
  --image-dir outputs/sd3_story/ablation_independent_scenes \
  --out-dir outputs/sd3_story/ablation_independent \
  --num-panels 3 --candidate 0
```

Public/original LoRA:

```bash
python scripts/sd3_infer.py \
  --prompts data/sd3_story/validation_prompts/prompts_3scene.jsonl \
  --out-dir outputs/sd3_story/public_lora \
  --lora-path artifacts/loras/sd3/public_lora \
  --lora-scale 0.8 \
  --width 1024 --height 768 --num-candidates 4
```

Self-trained Storyboard LoRA:

```bash
python scripts/sd3_infer.py \
  --prompts data/sd3_story/validation_prompts/prompts_3scene.jsonl \
  --out-dir outputs/sd3_story/self_lora \
  --lora-path artifacts/loras/sd3/storyboard_lora \
  --lora-scale 0.8 \
  --trigger-prefix sks_storyboard_style \
  --width 1024 --height 768 --num-candidates 4
```

Storyboard + Character LoRA:

```bash
python scripts/sd3_infer.py \
  --prompts data/sd3_story/validation_prompts/prompts_3scene.jsonl \
  --out-dir outputs/sd3_story/self_lora_character \
  --lora-path artifacts/loras/sd3/storyboard_lora \
  --lora-path artifacts/loras/sd3/character_lora \
  --lora-scale 0.8 \
  --lora-scale 0.8 \
  --trigger-prefix "sks_storyboard_style, sks_storyhero" \
  --width 1024 --height 768 --num-candidates 4
```

Scale sweep:

```bash
for scale in 0.4 0.6 0.8 1.0; do
  python scripts/sd3_infer.py \
    --prompts data/sd3_story/validation_prompts/prompts_3scene.jsonl \
    --out-dir outputs/sd3_story/self_lora_scale_${scale} \
    --lora-path artifacts/loras/sd3/storyboard_lora \
    --lora-scale $scale \
    --trigger-prefix sks_storyboard_style \
    --width 1024 --height 768 --num-candidates 4
done
```

## 7. Evaluation

Resolution check:

```bash
python scripts/sd3_check_resolution.py --dir outputs/sd3_story/self_lora --min-short 512 --fail-on-bad
```

CLIPScore:

```bash
python scripts/sd3_eval_clipscore.py \
  --image-dir outputs/sd3_story/self_lora \
  --prompts data/sd3_story/validation_prompts/prompts_3scene.jsonl \
  --out outputs/sd3_story/metrics/self_lora_clipscore.csv
```

DreamSim:

```bash
python scripts/sd3_split_panels.py \
  --image-dir outputs/sd3_story/self_lora \
  --out-dir outputs/sd3_story/panels_split/self_lora \
  --num-panels 3

python scripts/sd3_eval_dreamsim.py \
  --panel-dir outputs/sd3_story/panels_split/self_lora \
  --out outputs/sd3_story/metrics/self_lora_dreamsim.csv
```

Human evaluation sheet:

```bash
python scripts/sd3_make_human_eval_sheet.py \
  --image-dirs outputs/sd3_story/base_sd3 outputs/sd3_story/ablation_independent outputs/sd3_story/public_lora outputs/sd3_story/self_lora outputs/sd3_story/self_lora_character \
  --out data/sd3_story/human_eval/human_eval_sheet.csv
```

After evaluators fill the sheet:

```bash
python scripts/sd3_summarize_human_eval.py \
  --sheet data/sd3_story/human_eval/human_eval_sheet.csv \
  --out outputs/sd3_story/metrics/human_eval_summary.csv
```

Report assets:

```bash
python scripts/sd3_collect_report_assets.py
```
